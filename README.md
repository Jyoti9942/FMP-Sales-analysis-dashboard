# FMP-Sales-analysis-dashboard

This project presents an interactive Sales Analysis Dashboard built using Excel, Pivot Tables, Pivot Charts, and Slicers to analyze sales performance for FNP.
The dashboard provides clear insights into revenue trends, customer behavior, product performance, and monthly order patterns to support data-driven decision-making.
